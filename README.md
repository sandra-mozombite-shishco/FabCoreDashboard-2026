# FabCore Dashboard 2026

Interactive dashboard for managing academic manufacturing projects at FabCore Lab.

## Tech stack
- Quarto
- Python
- Git + GitHub

## Requirements
- Quarto (portable)
- Python 3.14
- VS Code

## How to run
1. Open folder in VS Code
2. Ensure Quarto is on PATH (workspace settings)
3. Press `Ctrl + Shift + B` or click Render

## Notes
- No admin permissions required
- Tools live in `D:\Portables`
